Bid Pilot

Finish setup

Bid Pilot drafts proposals through the Claude CLI on your computer, so there is no API key and nothing is billed per use. Browsers don't let an extension install that bridge by itself, so this one step links them. Takes about 20 seconds.

Checking connection…

  1. Before you start

    You need Node.js and the Claude Code CLI installed and logged in. Quick check in a terminal: echo hi | claude -p should print a reply.

  2. 1 · Download the installer

    One file, pre-filled with this extension's ID. It installs the bridge into every browser it finds.

  3. 2 · Run it once

    Open a terminal and paste this:

    bash ~/Downloads/bidpilot-install.sh

    Reads as one line. It writes the bridge to your home folder and registers it with your browsers. Safe to re-run.

  4. 3 · You're done

    This page turns green automatically once it connects. If it doesn't within a few seconds, fully quit and reopen your browser.

Why a terminal step?

Browser security forbids an extension from writing native-messaging files to disk. The downloaded script does exactly that one job: it drops a small Node host and registers it so the extension can talk to your local claude CLI. Nothing leaves your machine.
